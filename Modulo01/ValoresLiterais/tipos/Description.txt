
Plataforma .Net
O framework .net é uma plataforma, mas tambem é uma maquina virtual igual ao JVM do java. 
Quando compilo algo no C#, ele gera um programa em ByteCode. 
que é uma linguagem intermediaria que é lida pela maquina virtual do .Net Framework
